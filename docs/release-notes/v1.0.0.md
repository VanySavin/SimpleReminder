# SimpleReminder v1.0.0

Первый публичный релиз — portable-версия для Windows x64.

## Скачать

| Файл | Описание |
|------|----------|
| **SimpleReminder-v1.0.0-win-x64-portable.zip** | Распакуйте и запустите `SimpleReminder.exe` (~68 МБ, .NET внутри) |

Установка .NET на компьютере **не требуется**.

## Что умеет

- Напоминания с вводом на русском: `через 5 минут`, `завтра в 9`, `в понедельник в 23 отчёт` и др.
- Список активных задач, отложить, выполнено, удалить.
- Всплывающие уведомления, звук, очередь.
- Тема, позиции окон, кнопки отложения, автозапуск с Windows.
- Работа в системном трее.
- Данные локально в папке `data/` рядом с exe.

## Как запустить

1. Скачайте и распакуйте zip в любую папку.
2. Запустите `SimpleReminder.exe`.
3. При первом запуске Windows может показать SmartScreen — **Подробнее** → **Выполнить в любом случае**.
4. Если файл скачан из браузера: ПКМ по exe → **Свойства** → **Разблокировать**.

Папка `data/` создаётся автоматически — не удаляйте её для сохранения напоминаний.

## Системные требования

- Windows 10 или 11 (64-bit)
- ~80 МБ свободного места на диске

## Изменения

- Первый релиз исходников и portable-сборки.
