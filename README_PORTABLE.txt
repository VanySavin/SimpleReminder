SimpleReminder
Локальные напоминания для Windows.

Как запустить:
1. Распакуйте папку в любое место.
2. Запустите SimpleReminder.exe.
3. Папка data создаётся при первом запуске — не удаляйте её, если хотите сохранить напоминания и настройки.

Важно:
- Программа portable — установка .NET не нужна.
- Запускайте SimpleReminder.exe из этой папки (~70 МБ). Если Windows просит установить «.NET Desktop Runtime», вы запустили не ту сборку (например, маленький exe ~170 КБ из папки разработки).
- Все данные хранятся рядом с exe в папке data (reminders.json, settings.json).
- Чтобы полностью перенести программу на другой компьютер, скопируйте всю папку целиком.
- Для выхода используйте иконку в трее — Выход.

Автозапуск:
- Включите в Настройках программы («Запускать вместе с Windows»).
- При переносе папки на другой диск или компьютер выключите автозапуск и включите снова — Windows запоминает путь к exe.

Примеры фраз для напоминаний:
- через 5 минут позвонить
- завтра в 9 суп
- в понедельник в 23 отчёт
- суп через 5 минут
- через 1 час 36 минут проверить почту

Если Windows не запускает программу с первого раза:
- При первом запуске Windows может показать предупреждение, потому что программа portable и пока без цифровой подписи.
- Нажмите «Подробнее».
- Нажмите «Выполнить в любом случае».
- После первого разрешения программа обычно запускается обычным двойным кликом.
- Если файл был скачан из интернета: правой кнопкой по SimpleReminder.exe -> Свойства -> Разблокировать -> ОК (если такой пункт есть).
- Полностью убрать такое предупреждение можно только цифровой подписью приложения.
