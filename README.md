# SimpleReminder

**Локальные напоминания для Windows** — пишете обычными словами, когда и о чём напомнить. Без облака и без обязательного интернета: всё хранится на вашем компьютере в папке рядом с программой.

| | |
|---|---|
| **Платформа** | Windows 10/11 (x64) |
| **Стек** | WPF, .NET 8 |
| **Режим** | Portable — установка .NET на целевом ПК не нужна (в готовой сборке) |
| **Язык ввода** | Русский (естественные фразы для времени) |

---

## Зачем нужна программа

SimpleReminder помогает не забывать дела: лекарства, звонки, встречи, оплату счетов или любой другой текст. Вы описываете **когда** и **что** напомнить одной строкой. Напоминания и настройки сохраняются локально в папке `data/` — при переносе на другой ПК копируете **всю папку** с программой целиком.

## Возможности

- **Ввод текстом на русском** — программа сама разбирает время и текст напоминания.
- **Неограниченный список** активных напоминаний.
- **Всплывающие уведомления** поверх других окон, с очередью и звуком (настраивается).
- **Отложить** на заданные интервалы — набор кнопок в настройках.
- **Системный трей** — работа в фоне, выход через иконку в трее.
- **Тема** (светлая/тёмная), позиции окон уведомлений, автозапуск с Windows.
- **Тест уведомления** — посмотреть, как будет выглядеть окно.

## Как это работает

1. Вводите одну строку: время своими словами, затем смысл напоминания.
2. Программа вычисляет дату и время срабатывания и добавляет задачу в список.
3. В нужный момент появляется окно (и при необходимости звук).
4. В окне можно отметить выполненным или отложить.
5. Главное окно можно свернуть — приложение остаётся в трее (см. настройки).

## Быстрый старт (для пользователей)

### Скачать и запустить

1. Скачайте **`SimpleReminder-v1.0.0-win-x64-portable.zip`** из [Releases](https://github.com/VanySavin/SimpleReminder/releases/latest) **или** соберите сами — раздел [Сборка portable](#сборка-portable-для-переноса-на-другой-пк) ниже.
2. Распакуйте в любое место на диске.
3. Запустите **`SimpleReminder.exe`** из папки `SimpleReminder-win-x64-portable`.
4. Папка **`data/`** создаётся при первом запуске — в ней `reminders.json` и `settings.json`. **Не удаляйте** её, если хотите сохранить напоминания.

> **Важно:** portable-версия — exe **~68–71 МБ** (.NET внутри). Если Windows просит установить «.NET Desktop Runtime», вы запустили **не ту** сборку (например, маленький exe **~170 КБ** из `bin\Release\` — он требует установленный .NET 8 на этом ПК).

Подробная инструкция для конечного пользователя: [`README_PORTABLE.txt`](README_PORTABLE.txt).

### Примеры фраз

| Фраза | Пояснение |
|-------|-----------|
| `через 5 минут позвонить` | Через 5 минут от сейчас |
| `завтра в 9 суп` | Завтра в 09:00 |
| `в понедельник в 23 отчёт` | Ближайший понедельник, 23:00 |
| `суп через 5 минут` | Время в конце — тоже поддерживается |
| `через 1 час 36 минут проверить почту` | Составное время |
| `послезавтра в 15:30 забрать посылку` | |
| `сегодня в 18:00 ужин` | |
| `в 14:30 совещание` | Сегодня, если время ещё впереди; иначе завтра |

Если время не распознано — сравните с примерами или слегка измените формулировку.

### Первый запуск Windows (SmartScreen)

Portable-приложение без цифровой подписи может вызвать предупреждение:

1. **Подробнее** → **Выполнить в любом случае**.
2. Если файл скачан из интернета: ПКМ по exe → **Свойства** → **Разблокировать** (если есть).

### Автозапуск с Windows

Включите в **Настройках** → «Запускать вместе с Windows». После переноса папки на другой диск выключите и включите снова — Windows запоминает путь к exe.

## Сборка и разработка

### Что нужно для сборки

| Инструмент | Зачем |
|------------|--------|
| **[.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)** | Обязательно для `dotnet build` / `dotnet publish` |
| **Visual Studio 2022** (опционально) | Удобная разработка WPF; workload «Разработка классических приложений .NET» |
| **PowerShell** | Скрипт `publish-portable-win-x64.ps1` |
| **Git** | Клонирование репозитория |

Проверка SDK:

```powershell
dotnet --version
```

Должна быть версия **8.x**.

### Сборка portable (для переноса на другой ПК)

**Не путать** с обычным Build в Visual Studio:

| Тип сборки | Папка | Размер exe | .NET на другом ПК |
|------------|-------|------------|-------------------|
| **Portable** | `publish\SimpleReminder-win-x64-portable\` | ~68–71 МБ | **Не нужен** |
| Release (dev) | `src\ReminderApp\bin\Release\net8.0-windows\` | ~170 КБ + dll | **Нужен** .NET 8 Desktop Runtime |

**Самый простой способ** — в корне репозитория:

```bat
build-portable.bat
```

Или PowerShell:

```powershell
.\publish-portable-win-x64.ps1
```

Перед сборкой закройте запущенный `SimpleReminder.exe`.

Результат:

```
publish\SimpleReminder-win-x64-portable\
  SimpleReminder.exe
  README.txt          (копия README_PORTABLE.txt)
  data\               (создаётся при первом запуске)
```

Подробнее по шагам и типичным ошибкам: [`КАК_СОБРАТЬ.txt`](КАК_СОБРАТЬ.txt).

#### Visual Studio (альтернатива)

1. Откройте `src\ReminderApp\ReminderApp.csproj`.
2. ПКМ по проекту → **Опубликовать (Publish)**.
3. Профиль: **`Portable-win-x64`** (`Properties\PublishProfiles\Portable-win-x64.pubxml`).

### Сборка для разработки (без portable)

```powershell
dotnet build src/ReminderApp/ReminderApp.csproj -c Release
dotnet run --project src/ReminderApp/ReminderApp.csproj
```

### Тесты

```powershell
dotnet test tests/ReminderApp.Tests -c Release
```

Unit-тесты покрывают парсер русских фраз для времени (`ReminderTextParser`).

## Структура репозитория

```
winproject/
├── src/ReminderApp/          # WPF-приложение
├── tests/ReminderApp.Tests/  # тесты парсера
├── tools/GenReminderIcon/    # утилита генерации иконки
├── build-portable.bat        # сборка portable одним кликом
```

Папки **`publish/`**, **`bin/`**, **`obj/`**, **`.vs/`**, **`.cursor/`**, **`data/`** в git не попадают (см. `.gitignore`).
